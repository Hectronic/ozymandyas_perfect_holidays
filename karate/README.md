# OpenWeatherMap Karate Tests

Karate DSL tests for OpenWeatherMap Current Weather and Forecast APIs.

## Prerequisites

* Java 17
* Maven
* (Optional) Docker

## Setup

```bash
git clone <repo-url>
cd ozymandyas_perfect_holidays/karate
```

## Run

Locally:

```bash
./run.sh
```

In Docker:

```bash
./run_docker.sh
```

## Structure

```
Dockerfile
run.sh
run_docker.sh
pom.xml
tests/
  schemas/
  *.feature
```
